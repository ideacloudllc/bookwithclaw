## Escalation / Blockers

### If Cold Emails Aren't Getting Responses
**Action:**
- Switch to phone-first outreach (more personal)
- Ask for referrals from first 2-3 sellers
- Post on LinkedIn/Twitter (thought leadership)
- Reach out to hotel industry forums/groups

### If Sellers Aren't Onboarding
**Action:**
- Simplify onboarding (remove unnecessary steps)
- Offer white-glove setup (do it for them)
- Create video walkthrough
- Reduce friction (pre-fill some fields)

### If First Buyer Isn't Ready
**Action:**
- Manually create test buyer intents (simulate demand)
- Partner with early buyer to do real test
- Start buyer recruitment in parallel

## Post-Signup Workflow

### What Happens After Someone Signs Up

1. **Immediately (auto-email)**
   - Confirmation email
   - Seller Onboarding Guide
   - Link to dashboard
   - 30-min onboarding call invite

2. **Within 2 hours (manual)**
   - Welcome call (confirm they're set up)
   - Answer questions
   - Create first room listing with them
   - Test AI negotiation with sample offer

3. **Within 24 hours**
   - Check in: Did they add photos? Set rates?
   - If stuck: Offer help
   - Create test buyer intent (show AI in action)
   - Get testimonial request

4. **Week 1 follow-up**
   - Are they seeing activity?
   - Any first real offers?
   - Are they comfortable with AI negotiations?
   - Request referral (for Phase 2 growth)
